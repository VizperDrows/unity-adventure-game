# unity-adventure-game

# Christmas Wonderland 

A festive Unity-based adventure game where players explore a winter world, collect gifts, and progress through escalating challenge stages with unique character abilities.

# Game Overview

In Christmas Wonderland, the player navigates an open environment collecting gifts scattered across the map. As the player progresses, the difficulty scales through a staged system where each phase requires exponentially more gifts.

A dynamic event system introduces Santa Claus as an announcer, guiding the player through each stage with contextual dialogue.

# Character System

Each character has unique abilities:

- Santa
  - Can fly freely
  - Ability to locate nearest gift via directional trail
-Trunks
  - Can deploy lightbulbs to decorate trees
  - Special interaction with environmental objects
  - Does not receive speed boosts from gifts
- Bobby
  - Can throw snowballs
  - Can spawn decoy snowmen
  - Melts when touching water
- Frank
  - Uses a gun to shoot projectiles
  - Can place bear traps
  - Alters enemy behavior (frenzy mode)
  - No Santa announcer interaction
- Rudolph
  - Speed-based abilities
- Elf
  - Standard movement-focused gameplay
